How delivery works
Rotation OFF
All mail goes through the DEFAULT server. Use its From Address (or message's From).
Rotation ON
Each message uses the next server in round-robin. Set From Address on each server so the correct sender is used.

Bulk Import SMTP Servers
One server per line. Format: host:port:user:pass:tls[:from@domain.com] — TLS: 3=auto (by port), 0=none, 1=TLS, 2=SSL

smtp.sendgrid.net:587:apikey:SG.xxxx:1:sender@yourdomain.com
smtp.example.com:465:user:pass:2
