Outbound IP Addresses

When enabled, outbound connections will rotate through the IPs listed below (round-robin).

Lines starting with # are ignored. First word on each line is taken as the IP.

How it works
  • Each outbound connection binds to the next IP in the list
  • Round-robin distributes load evenly
  • If binding fails, system falls back to default IP
  • Each IP should have a PTR/rDNS record pointing to your HELO name
  • Add each IP to your SPF record
SPF with multiple IPs
v=spf1 ip4:1.2.3.4 ip4:5.6.7.8 ~all
