Let's Encrypt (Recommended for Production) Free, trusted, auto-renewal

Let's Encrypt provides a free, publicly trusted certificate. Requirements:

  • Your server's hostname must point to this server's IP in DNS
  • Port 80 (HTTP) must be publicly accessible for the ACME challenge
  • Port 443 or 25 can be used after cert is issued
Run on your server (requires certbot):
certbot certonly --standalone -d {{.HeloName}} --agree-tos --non-interactive
Then update config.yaml with the issued cert paths:
smtp:
  tls:
    enabled: true
    cert_file: /etc/letsencrypt/live/{{.HeloName}}/fullchain.pem
    key_file: /etc/letsencrypt/live/{{.HeloName}}/privkey.pem
{{end}}
